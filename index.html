03. projects

Sitio oficial de la Unión Regional Deportiva

Se desarrolló un sitio completo, institucional e informativo, para la Unión Regional Deportiva de las ciudades de Tandil, Ayacucho, Rauch y Benito Juárez, utilizando metodología de desarrollo MVC.

También se implementó, y creó desde cero, un CMS para gestionar las noticias y administrar las tablas de posiciones.

Sistema estadístico para Sub-15

Este proyecto constó de contador de estadísticas para el cuerpo técnico de la Selección Sub-15 de la ciudad de Tandil.

Los principales lenguajes utilizados fueron HTML, CSS, JavaScript y posteriormente SQL para almacenar los datos recolectados en cada jornada.

E-commerce para ferretería local

Para satisfacer las necesidades del cliente y ante la necesidad de cubrir la ausencia en la venta online, se implementó una plataforma de e-commerce para la ferretería H30, en su sucursal para la ciudad de Tandil.

Se analizó la imagen de la marca, se presentaron propuestas de diseño y siguiendo el proceso productivo, se llevó a cabo el proyecto manteniendo un diálogo fluído con el cliente.

Juego “4 en Línea”

Como práctica universitaria, se diseñó y desarrolló una simulación de un sitio de videojuegos online para la cual se siguieron distintas leyes y fundamentos de UX y UI, adquiridas durante la carrera.

Además, dentro de la misma web, se creó el famoso juego "4 en Línea", totalmente funcional y con temática futbolera, implementando matrices y canvas con JavaScript.
